**Setup Environment Computer Vision**
**Tujuan**
Melakukan instalasi dan konfigurasi environment untuk praktikum Machine Learning khususnya Computer Vision.

**Tools yang Digunakan**
1. Anaconda
2. Python 3.10
3. Jupyter Lab
   
**Langkah-langkah**
1. Membuat environment baru: conda create -n mlcv python=3.10
2. Mengaktifkan environment: conda activate mlcv
3. Install library: pip install numpy matplotlib opencv-python
4. Menjalankan Jupyter Lab: jupyter lab

**Hasil**
Environment berhasil dibuat dan Jupyter Lab dapat dijalankan dengan kernel Python aktif.
